# cTerminal (Archived)

> AI-first multi-model terminal with Claude, Ollama, and MCP tool support.

## What This Was

A desktop terminal application that unified multiple AI models (Claude, Ollama) into a single interface with MCP tool integration. Built to explore what a native AI-first terminal experience could look like when models can invoke tools directly.

## Stack

- Tauri 2.0 (Rust backend)
- React (TypeScript frontend)
- Claude API + Ollama (local models)
- MCP (Model Context Protocol) tool system

## Usage

This project is archived and no longer maintained.
